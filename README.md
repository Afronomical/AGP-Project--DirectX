# Advanced Graphics Programming project
This project was my first time delving into DirectX 11 and 3D rendering as a whole. I created a game engine in DirectX 11. I created shaders, and managed buffers correctly and was able to create and handle a number of other key components of a game engine such as:
- Input
- Meshes
- Textures
- Lighting
- DeltaTime
 and more...


![](https://github.com/Afronomical/AGP-Project--DirectX/blob/main/Readme%20Visuals/AGP%20Demo%20scene%20gif.gif)

 # Installation & Usage
 - Download the github files namely: ```AGP DX11 File```
 - Open the file called ```GraphicsRendererC++.sln``` using visual studio or an appropriate code editor
 - Press Run, or press F5 (if in visual studio)

![](https://github.com/Afronomical/AGP-Project--DirectX/blob/main/Readme%20Visuals/AGP%20Planets%20GIF.gif)
  ## Changing Scene
 - In order to change the scene between the two demo scenes open ```Program.h```
 - Within the solution explorer locate the ```currentScene``` variable.
 - Uncomment the scene you would like to play in it's initialisation
 - Comment out the alternative
